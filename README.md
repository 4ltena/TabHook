# TabHook

放置されたタブの記憶を呼び起こすChrome拡張機能。
A Chrome extension that reignites your memory of abandoned tabs.

Version 0.9.5

## 日本語

### これは何？
「あのタブ、何だったっけ？」——放置されたタブの内容をLLMが分析。あなたの好奇心をそそる断片をデスクトップへ届けます。続き、気になりませんか？

### 主な機能
- **好奇心を刺激する通知**: 単なる要約ではなく、思わず戻りたくなるような一言を生成。
- **複数のモデルに対応**: OpenAI, Claude, Geminiなどの公開モデルに加え、OllamaによるローカルLLMにも対応。
- **リソース管理**: ローカルLLM使用時のVRAM解放ボタンを完備。ポート番号も自由に変更可能。
- **プライバシー配慮**: ローカルLLMモードなら、外部サーバーにデータを送らずに要約が可能。

### 使い方
1. 拡張機能をインストールし、設定画面（Popup）を開く
2. AIモード（公開モデル or ローカルLLM）を選択
3. 放置判定までの時間（分）を設定
4. 指定時間の旅を終えると、AIがあなたの再訪を促します。

## English

### What is it?
"What was that tab about?" — LLM analyzes your abandoned tabs and delivers intriguing snippets straight to your desktop. Curious?

### Key Features
- **Curiosity-Driven Notifications**: Generates snippets that make you want to dive back in.
- **Multiple Models Support**: Supports major Public LLMs (OpenAI, Claude, Gemini) and Local LLMs via Ollama.
- **Resource Management**: Includes a "Release VRAM" button for local LLM users. Custom port configuration is also supported.
- **Privacy First**: With Local LLM mode, your content stays on your machine.

### How to Use
1. Install the extension and open the Settings (Popup)
2. Choose your AI Mode (Public Models or Local LLM)
3. Set the "Timer Duration" (minutes)
4. Once the tab completes its journey through time, AI will invite you back.
